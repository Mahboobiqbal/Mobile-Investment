# Frontend Structure

This folder contains the entire React Native app.

## Key areas
- `src/components` for reusable UI
- `src/features` for app modules
- `src/navigation` for app routing
- `src/store` for global state
- `src/context` for React Context providers
- `src/services` for API and storage layers
- `src/utils` for helpers and validators
- `src/theme` for design tokens
- `src/types` for shared TypeScript definitions
